# SuperPro Designer

A standalone bioprocess flowsheet studio inspired by process simulation tools.

## Open locally

Open `index.html` in a browser.

## What is included

- Process templates for cultured meat, penicillin, monoclonal antibody, and industrial fermentation production
- Scale presets for lab, pilot, demo, and commercial designs
- Click-to-add and drag-and-drop unit-operation library with reactors, fermenters, filters, chromatography, UF/DF, crystallizers, dryers, filling, and CIP
- Moveable flowsheet nodes, copy selected unit, and connect mode for creating new process streams
- Live batch volume, annual batch, titer, recovery, production, utilization, utility, and direct-cost estimates
- Editable unit and stream inspector
- Equipment and stream tables
- Searchable equation library for stoichiometry, kinetics, mass balances, energy balances, separations, and economics
- Cost-driver economics view
- JSON scenario export

No package install is required.
