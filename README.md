# SuperPro Designer

A standalone bioprocess flowsheet studio inspired by process simulation tools.

## Open locally

Open `index.html` in a browser.

## What is included

- Process templates for cultured meat, penicillin, monoclonal antibody, industrial fermentation, recombinant insulin, viral vaccine, plasmid DNA, autologous cell therapy, and small-molecule API production
- Scale presets for lab, pilot, demo, and commercial designs
- Click-to-add and drag-and-drop unit-operation library with ISO/PFD-style pharmaceutical equipment names, icons, and GMP/ICH/ISO metadata
- Moveable flowsheet nodes, copy selected unit, connect mode, auto-layout, fit-to-view, and zoom controls for large process flows
- Live batch volume, annual batch, titer, recovery, production, utilization, utility, and direct-cost estimates
- Editable unit and stream inspector
- Equipment and stream tables
- Searchable equation library for stoichiometry, kinetics, mass balances, energy balances, separations, and economics
- Standards library covering EU GMP Annex 1/15, FDA 21 CFR, ICH Q-series guidance, ISO 14644, ISO 13408, ISO 10628, ASME BPE, ISA-88/95, GAMP 5, USP chapters, and ISO 15378
- Cost-driver economics view
- JSON scenario export

No package install is required.
