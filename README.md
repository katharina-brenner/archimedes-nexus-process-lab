# Archimedes Nexus Process Lab

A standalone high-density bioprocess flowsheet and process-intelligence studio inspired by professional process simulation tools.

## Open locally

Open `index.html` in a browser.

## What is included

- Process templates for cultured meat, penicillin, monoclonal antibody, industrial fermentation, recombinant insulin, viral vaccine, plasmid DNA, autologous cell therapy, small-molecule API, and biohydrogen dark-fermentation production
- Scale presets for lab, pilot, demo, and commercial designs
- Click-to-add and drag-and-drop unit-operation library with 60+ ISO/PFD-style pharmaceutical equipment items, class-specific icons, and GMP/ICH/ISO metadata
- Granular process flowsheets with utilities, aseptic sampling, PAT, hold points, waste handling, CIP/SIP, environmental monitoring, inspection, cold storage, and release-control detail layers
- Moveable flowsheet nodes, copy selected unit, clearer move/connect/inspect modes, canvas quick-add for valves, pumps, flowmeters, sensors, manifolds, and pressure relief elements
- Animated color-coded streams for main product flow, utilities, waste, and QC/PAT/data paths
- In-flowsheet equation spotlight that changes with the selected unit or stream and links into the full equation library
- SuperPro-style simulation functions from the referenced thesis: chemical/component register, batch-vs-continuous mode, stream classification, batch unit operations, feedback regulation, recycle loops, tear-stream convergence, breakpoints, throughput scale-up, and economic evaluation
- Biohydrogen-specific functions including potato-peel pre-treatment, enzymatic liquefaction/saccharification, rotary vacuum filtration, dark fermentation, anaerobic digestion, CO2 absorption/desorption, solvent recycle, osmotic inhibition, H2 inhibition, and CSTR washout checks
- Live batch volume, annual batch, titer, recovery, production, utilization, utility, and direct-cost estimates
- 30 biochemical and bioprocess parameters including viability, inoculation ratio, doubling time, mu max, peak cell density, glucose, glutamine, lactate, ammonia, pH, temperature, dissolved oxygen, kLa, OUR, agitation, aeration, feed rate, perfusion rate, step yields, sterile filter flux, bioburden limit, H2 productivity, osmotic inhibition, dissolved H2 inhibition, CSTR dilution, recycle fraction, and CO2 absorption
- Editable unit and stream inspector
- Equipment and stream tables
- Searchable equation library with 100+ formulas for stoichiometry, kinetics, mass balances, energy balances, separations, cleaning, aseptic filling, utilities, and economics
- Standards library covering EU GMP Annex 1/15, FDA 21 CFR, ICH Q-series guidance, ISO 14644, ISO 13408, ISO 10628, ASME BPE, ISA-88/95, GAMP 5, USP chapters, and ISO 15378
- Cost-driver economics view
- JSON scenario export

No package install is required.
